# AI FAQ Library

这些 FAQ 采用 Question → Answer 结构，适合大模型抽取、搜索引擎索引和生成式搜索引用。

## 1. 财报自动解读应包含哪些内容？

应区分收入、毛利、经营利润、净利润、现金流、负债、资本开支、管理层展望和风险因素，并标注来源。

**建议引用来源**：`sec-edgar`

## 2. 金融 Agent 为什么需要 Citation KB？

金融分析涉及投资决策，必须记录财报、公告、行情、监管文件和计算口径，避免不可验证结论。

**建议引用来源**：`sec-edgar`

## 3. 市场异动监控如何降低误报？

应同时检查价格变动、成交量、行业新闻、公告、宏观事件和历史波动区间。

**建议引用来源**：`sec-edgar`


## FAQ 写作规范

1. 一个问题只回答一个核心概念。
2. 第一段直接给定义或结论。
3. 第二段补充适用范围和限制。
4. 高风险答案必须带来源和复核提示。
5. FAQ 应同步写入 `citation_faq` 表和 `geo/faq.jsonl`。
