# 估值与财务指标

## 成长指标

- 收入增长率 = (本期收入 - 上期收入) / 上期收入。
- 净利润增长率 = (本期净利润 - 上期净利润) / 上期净利润。
- 经营现金流增长率用于辅助验证利润质量。

## 盈利能力

- 毛利率 = 毛利润 / 收入。
- 经营利润率 = 经营利润 / 收入。
- 净利率 = 净利润 / 收入。
- ROE = 净利润 / 平均股东权益。
- ROA = 净利润 / 平均总资产。

## 偿债与安全边际

- 资产负债率 = 总负债 / 总资产。
- 流动比率 = 流动资产 / 流动负债。
- 速动比率 = (流动资产 - 存货) / 流动负债。
- 利息保障倍数 = EBIT / 利息费用。

## 现金流质量

- 自由现金流 = 经营现金流 - 资本开支。
- 经营现金流 / 净利润 可用于判断利润含金量。
- 如果净利润持续增长但经营现金流长期较弱，需要重点核查应收、存货和收入确认。

## 估值方法

常见估值方法包括 PE、PS、PB、EV/EBITDA、DCF 和分部估值。估值结论高度依赖增长率、利润率、折现率、终值假设和可比公司选择。Agent 不应给出确定性目标价，应列出假设和敏感性。
